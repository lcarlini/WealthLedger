Get started

Install .NET 10 and Node.js 20+ once, then run two PowerShell scripts. Everything stays inside the project folder — nothing is installed elsewhere on your machine.

Get the code

Download or clone the project, then open a terminal in its folder.

git clone <your-fork-url> WealthLedger
cd WealthLedger

Set it up

Checks your tools, downloads the local AI model, and builds the app.

./deploy/1-setup.ps1

Run it

Starts the app and opens it in your browser automatically.

./deploy/2-run.ps1

The app opens at http://localhost:5000 — one local server for both the app and the API. On Windows you can also just right-click a script and choose “Run with PowerShell”.
