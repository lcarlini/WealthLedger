Get started

Requires the .NET 10 SDK and Node.js 20+.

Clone & install

Grab the code and install the frontend dependencies.

git clone <your-fork-url> WealthLedger
cd WealthLedger/Source/Run/SPA
npm ci

Run the API

Starts the backend on port 5000. The database is created automatically.

cd Source/Run/WebApp
dotnet run

Run the app

Launch the UI on port 4200, then open your browser.

cd Source/Run/SPA
npm start

Open http://localhost:4200 — or see the full installation, deployment & backup guide.
